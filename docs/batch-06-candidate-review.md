# Batch-06 候補レビュー

Stage9では `batch-06` の本実装は行わず、候補30〜40件をカテゴリ別に整理します。選定基準は、既存100モチーフとのrelationを説明できること、1文字aliasに依存しないこと、質問とblockで意味差分を作れることです。

## 候補一覧

| category | candidates | note |
|---|---|---|
| 場所 | レストラン、カフェ、コンビニ、映画館、温泉、交差点 | 生活圏と移動先を補強 |
| 物・道具 | 窓、机、椅子、眼鏡、イヤホン、地図、切符、チケット | 1文字aliasを避けやすく、文脈phrase化しやすい |
| 行動 | 走る、掃除する、片付ける、失くす、壊す、渡す | 活用形監査が必要だが、質問差分を作りやすい |
| 自然・生き物 | 虫、馬、雲、虹、砂、石 | 水/天候以外の自然象徴を補強 |
| 身体・感覚 | 手、足、眠る、痛み | 高刺激・診断的表現に注意 |
| 対人・関係 | 上司、先生、子ども、同僚 | 既存のfriend/family/workplace/schoolとの重複を慎重に見る |
| 現代生活 | ゲーム、動画、充電、カレンダー、ニュース | 通信系と重なりすぎないものを優先 |

合計候補: 39件

## Stage10の絞り込み基準

- 既存100件とのrelationを3件以内で説明できる。
- phraseを最低3件作れる。
- 2文字以下weakに頼らない。
- 質問で少なくとも3択の意味差分が出る。
- 高刺激・診断的な文面にならない。
- Stage9 audit fixtureに近い誤検出リスクを事前に追加できる。
