# Batch-07 候補レビュー

Stage11では `batch-07` の本実装は行わず、候補30〜40件をカテゴリ別に整理します。選定基準は、既存120モチーフとのrelationを説明できること、1文字aliasに依存しないこと、質問とblockで意味差分を作れることです。

## 選定基準

- 既存120件とのrelationが3件以内で説明できる
- phraseを最低3件設計できる
- 2文字以下のweak語に依存しない
- 質問で3択以上の意味差分が出る
- 身体的・医療的断定を避けるblockトーンを設計できる
- 既存 fixture に誤検出を生まないことを事前 audit fixture で確認できる

## 見送り候補と理由

| id案 | 表示名 | 見送り理由 |
