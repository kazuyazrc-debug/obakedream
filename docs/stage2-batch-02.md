# Stage2 Batch-02 Notes

Stage2では、30から40候補を見直し、既存20モチーフとの重複と抽象度を確認したうえで20件を採用しました。

## 採用20モチーフ

- 火: 感情の勢い、浄化、怒りを扱える
- 雨: 水と近いが、感情の放出や足止めに特化できる
- 空: 視野、希望、俯瞰の読みを補える
- 月: 直感、周期、静かな感情変化を扱える
- 森: 内面探索、迷い、回復の場所として使いやすい
- 山: 目標、課題、成長の象徴として汎用性が高い
- 川: 水より具体的に流れと移行を扱える
- 橋: 境界を越える、関係をつなぐ読みを作れる
- 道: 進路、選択、迷いに意味を絞って採用
- 階段: 段階的変化と上昇下降を扱える
- エレベーター: 急な上下変化や他力で運ばれる感覚を扱える
- 車: 自分で進む力とコントロール感を扱える
- 職場: 会社より広く、仕事上の役割と評価を扱える
- 病院: 回復、点検、ケアの文脈を扱える
- 試験: 学校より具体的に評価イベントを扱える
- トイレ: 手放し、浄化、隠したい感情を扱える
- 鏡: 自己認識と見られ方を扱える
- 鍵: 解決の手がかり、境界、秘密を扱える
- 財布: 価値観、安心材料、喪失不安を扱える
- 元恋人: 過去の関係と未整理感を扱える

## 見送り候補

- 会社: 職場と重複するため、Stage2では職場に統一
- 職場: 採用
- 雪: 雨と近く、Stage3で季節/冷却系として再検討
- 風: 抽象度が高く、天候系が増えすぎるため見送り
- 太陽: 空/月との対比でStage3候補
- バス: 電車/車と近く、乗り物カテゴリ拡張時に検討
- 自転車: 車より個人的移動に寄せられるためStage3候補
- 船: 海と組み合わせる移動象徴としてStage3候補
- 旅行: 抽象度が高く、乗り物/場所を増やした後に検討
- 迷子: 道/森/知らない人と重なるため見送り
- 風呂: 水/トイレと近く、浄化系が増えすぎるためStage3候補
- スマホ: 現代的対人象徴として有力だが、alias設計を詰めてStage3へ
- 手紙: メッセージ系カテゴリを作る時に検討
- 食べる: 行動カテゴリ拡張時に検討
- 泣く: 雨/感情表現と近く、感情行動バッチで検討
- 笑う: 感情行動バッチで検討
- 怒る: 火/戦うと近く、感情行動バッチで検討
- 好きな人: 元恋人と恋愛系が広がりすぎるためStage3へ
- 告白: 恋愛派生としてStage3へ
- 結婚: 人生イベント系としてStage3へ
- 妊娠: 赤ちゃんと近く、ライフイベント拡張時に検討
